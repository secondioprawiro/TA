Repository ini berisi *source code* untuk aplikasi **Virtual Classroom** yang dikembangkan sebagai proyek Tugas Akhir (Skripsi).

Aplikasi ini dirancang untuk mengatasi keterbatasan pembelajaran jarak jauh konvensional (seperti Zoom/Meet) dengan menghadirkan **lingkungan 3D imersif**. Di sini, pengajar dan siswa dapat bertemu, berinteraksi, dan melakukan simulasi pembelajaran dalam satu ruangan virtual yang sama secara *real-time*, layaknya berada di kelas sungguhan.

## ğŸš€ Fitur Utama

Aplikasi ini memiliki beberapa kapabilitas utama:

* **Multi-User System:** Mendukung banyak pengguna (Siswa & Guru) untuk masuk ke dalam satu ruangan kelas secara bersamaan.
* **Multi-Platform:** Dapat diakses melalui perangkat **Desktop (PC/Laptop)** dan **Mobile (Android)** secara lintas platform (*cross-platform*).
* **Real-Time Interaction:** Sinkronisasi pergerakan avatar dan interaksi objek antar pengguna yang terjadi secara langsung tanpa tunda.
* **Materi Pembelajaran Interaktif:** Dilengkapi dengan simulasi alat peraga visual untuk membantu pemahaman materi (studi kasus: Fisika).
